おすすめのプロジェクト

Microsoft MakeCode

https://github.com/microsoft/pxt

Microsoft Programming Experience Toolkit (PXT)に基づいたオープンソースのプロジェクトです。
いわゆるScratchのようなブラウザ内にエディタとプログラミング実行環境があり、初心者向けの特別な目的のプログラミング体験を作成するためのフレームワークです。

Microsoft MakeCode Arcade

https://github.com/microsoft/pxt-arcade

上記のMakeCodeを電卓ほどの小さな端末でも動かすことができるプロジェクトです。
マイコンごとのソースコードとハードウェア(アートワークと回路図)を公開している。
